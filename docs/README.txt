Aleph One is the open source continuation of Bungie™'s Marathon® 2 and Marathon
Infinity game engines. Aleph One plays Marathon, Marathon 2, Marathon Infinity,
and third-party content on a wide array of platforms.

Binary releases and scenario files are available at the Aleph One web site:

<https://alephone.lhowon.org/>

Source code and instructions for compiling are available from GitHub:

<https://github.com/Aleph-One-Marathon/alephone>
